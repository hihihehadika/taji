# Tajilang (Ekstensi VS Code)

Dukungan resmi penyorotan sintaksis dan identitas visual untuk bahasa pemrograman Taji di Visual Studio Code.

## Fitur Utama
- **Penyorotan Sintaksis**: Mewarnai kata kunci, fungsi bawaan, string, dan komentar sesuai standar Taji.
- **Identitas Visual**: Ikon berkas khusus untuk setiap file berakhiran `.tj`.
- **Snippets & Autocomplete**: Penutupan kurung otomatis dan dukungan struktur bahasa Indonesia.

## Cara Instalasi
Karena ekstensi ini didistribusikan secara manual dalam format `.vsix`, ikuti langkah-langkah berikut untuk memasangnya:

1. Buka **Visual Studio Code**.
2. Buka bilah **Extensions** (Ikon kotak di sisi kiri atau tekan `Ctrl+Shift+X`).
3. Klik ikon **Titik Tiga (...)** (More Actions) di pojok kanan atas panel Extensions.
4. Pilih menu **Install from VSIX...**.
5. Cari dan pilih berkas `tajilang.vsix` yang ada di dalam folder ini.
6. Klik **Install**.
7. Restart VS Code jika diperlukan.
